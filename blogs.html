Question-1: What are the different ways to select an element in the DOM?

getElementById(' ') is more common to select an element in DOM.
Others are use occasionally, other options include getElementByTagName(' '), getElementByClassName(' '), querySelector(' '), and querySelectorAll(' ').
For the last two options, we need to use symbols in the brackets, for the class name, we need to use ('.className') and for the ID ('#idName').

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main difference between .innerText and .innerHTML is that. .innerText is only used to show the text content of an element, while .innerHTML is used to show all the HTML content including all the HTML tags within an element. Both innerText and textContent will ignore the HTML tags and show everything as a string.
innerText = only show the text;
innerHTML = all HTML code;
